// what it does

Your buddy, always on screen.

🎭
7 Animated Characters updated
Wall·E, Pikachu, Luffy, Zoro, Chopper, Po, and Riri — each with unique idle, click, celebrate, and 3 right-click personality animations.
✅
World-class Todo Panel updated
Filter by priority and type, search, sort, group by date (Today / This week / Older), done accordion, and a consistent task count badge inside and out.
🔔
Smart Notifications new
A floating bubble appears above your character with priority-aware auto-dismiss. High-priority items stay until you act on them.
🤖
Claude Code Integration new
Claude Code permission requests pop up as bubbles. Allow or Deny directly from Kaaku — no terminal needed. Bidirectional via Unix socket.
💬
Slack Sync new
Scans all your Slack channels for @mentions and DMs. AI classifies tasks, blockers, and deadlines. Resolves real display names — no raw user IDs.
🖱️
Drag, Rotate, Interact
Drag to move anywhere on screen — the panel auto-repositions to stay fully visible, even from corners. Left-drag to rotate 360°. Right-click for personality animations.
🚀
Launches at Login
Auto-starts silently when you log in — no dock icon, always floating, always on top of every window.
⚡
System Event Bus new
Unix domain socket at /tmp/kaaku.sock lets any tool or script push notifications to Kaaku. One CLI command away.
🦴
Skeletal GLB Animations
Characters use real rigged GLB models from Blender with smooth skeletal animation blending — not just procedural transforms.
// smart bubbles
